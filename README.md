# Sistema Experto para la Recomendación de Metodologías de Desarrollo de Software

Este proyecto consiste en un sistema experto desarrollado en Python con interfaz gráfica en Tkinter. Su objetivo es recomendar metodologías de desarrollo de software de acuerdo con las características de un proyecto.

## Características principales

- Modo libre tipo chat.
- Modo guiado mediante preguntas.
- Recomendación de metodologías: Scrum, Kanban, Cascada, XP, Lean, Espiral, Incremental, RAD y SAFe.
- Motor de inferencia basado en reglas y puntajes.
- Detección de condiciones mediante lenguaje natural.
- Validación de contradicciones.
- Preguntas de seguimiento.
- Gráfica de puntajes.
- Colores personalizados por metodología.
- Exportación de reportes a Word.

## Requisitos

- Python 3
- Tkinter
- python-docx
